extract_struc.py

Script to extract (a) structure(s) from init_struc_data.pkl and opt_struc_data.pkl, and output (a) cif file(s).

One can specify structure ID(s) using -i option. -a option is for outputting all the structures. (note that many cif files will be output.) -s option adds symmetry information, but note that

Usage

usage: extract_struc.py [-h] [-a] [-i [INDEX ...]] [-s] infile

positional arguments:
  infile                input file

optional arguments:
  -h, --help            show this help message and exit
  -a, --all_id          flag for all structures
  -i [INDEX ...], --index [INDEX ...]
                        structure ID
  -s, --symmetrized     flag for symmetrized structure

Examples

Structure ID(s)

$ python3 extract_struc.py init_struc_data.pkl -i 7 10 12

7.cif, 10.cif, and 12.cif are output.

Structure ID(s) with symmetry information

$ python3 extract_struc.py init_struc_data.pkl -i 7 10 12 -s

All the structures

You should make a directory.

$ mkdir init_cifs
$ cd init_cifs
$ python3 extract_struc.py init_struc_data.pkl -a

All the structures with symmetry information

$ mkdir init_cifs
$ cd init_cifs
$ python3 extract_struc.py init_struc_data.pkl -as
