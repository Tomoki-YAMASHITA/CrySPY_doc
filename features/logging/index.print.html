Logging

2023 July 10

CrySPY 1.2.0 adopts logging library of Python. CrySPY logs are output to both the screen and files(log_cryspy and err_cryspy).

  • log –> screen and log_cryspy
  • error and warning –> screen and err_cryspy

Here is the example:

If you want to run cryspy as a background job, or if you use the auto script, and do NOT want it to output to the screen, execute it with the -n option as follow:

cryspy -n
