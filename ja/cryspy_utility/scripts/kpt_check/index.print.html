kpt_check.py

kpt_check.pyはkppvolを入力した際にkメッシュがどれくらいになるかをチェックすることができる． POSCAR，CONTCAR，およびinit_struc_data.pklが読み込み可能． pymatgenライブラリが必要．

初期構造を生成後に，kppvolの値をどれくらいにしたら良いか色々試すことができる．（下記の最後の例を参照）

使い方

python3 kpt_check.py [-h] [-w] [-n NSTRUC] infile kppvol

positional arguments:
  infile                input file: POSCAR or CONTCAR or init_struc_data.pkl
  kppvol                kppvol

optional arguments:
  -h, --help            show this help message and exit
  -w, --write           write KPOINTS
  -n NSTRUC, --nstruc NSTRUC
                        number of structure to check

例

POSCARとkppvol

$ python3 kpt_check.py POSCAR 100
a = 10.689217
b = 10.689217
c = 10.730846
    Lattice vector
10.689217 0.000000 0.000000
0.000000 10.689217 0.000000
0.000000 0.000000 10.730846

kppvol:  100
k-points:  [2, 2, 2]

KPOINTSファイルの生成

POSCARおよびCONTCARを入力とする際は，-wオプションを使用することでKPOINTSファイルを生成できる．

$ python3 kpt_check.py -w POSCAR 100
$ cat KPOINTS
pymatgen 4.7.6+ generated KPOINTS with grid density = 607 / atom
0
Monkhorst
2 2 2

init_struc_data.pklとkppvol

init_struc_data.pklの初期構造データに任意のkppvolの値を適用した場合のkメッシュを調べることが可能． デフォルト設定でははじめの5構造をチェックする． -nを使うとチェックする構造数も変えられる．
