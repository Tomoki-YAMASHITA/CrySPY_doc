extract_struc.py

init_struc_data.pklやopt_struc_data.pklから構造ごとにcifファイルを出力するスクリプト．

-iオプションでインデックス（構造ID)を（複数）指定できる．-aオプションの場合は全ての構造をcifファイルに出力する．数が多い場合はディレクトリを作成してそこに移動してから実行したほうが良い．-sオプションをつけると対称性（空間群）の情報を付けられる．ただし，セルや原子数が変わるので注意．

使い方

usage: extract_struc.py [-h] [-a] [-i [INDEX ...]] [-s] infile

positional arguments:
  infile                input file

optional arguments:
  -h, --help            show this help message and exit
  -a, --all_id          flag for all structures
  -i [INDEX ...], --index [INDEX ...]
                        structure ID
  -s, --symmetrized     flag for symmetrized structure

例

構造ID指定

$ python3 extract_struc.py init_struc_data.pkl -i 7 10 12

以下のファイルが出力される．

7.cif，10.cif，12.cif

対称性の情報あり

$ python3 extract_struc.py init_struc_data.pkl -i 7 10 12 -s

全構造

構造が多い場合はディレクトリを先に作った方が良い．

$ mkdir init_cifs
$ cd init_cifs
$ python3 extract_struc.py init_struc_data.pkl -a

全構造（対称性あり）

$ mkdir init_cifs
$ cd init_cifs
$ python3 extract_struc.py init_struc_data.pkl -as
