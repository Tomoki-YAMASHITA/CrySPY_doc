システム要件

使用予定のワークステーションやスパコンにおいて，(CrySPY + Python環境 + 構造最適化ソフト) が必要となる．

Python

Tested with Python 3.7.5 and 3.8.5 on Mac, and Python 3.6.7 (miniconda3-4.3.30) and 3.8.2 on Linux.

  • Python 3.x.x
    • COMBO
    • pymatgen
    • PyXtal (CrySPY 0.9.0 以上)

構造最適化ソフト

少なくとも１つは必要．

  • 第一原理計算
    • VASP
    • Quantum Espresso
    • OpenMX (CrySPY 0.9.0 以上)
  • 原子間ポテンシャル
    • soiap
    • LAMMPS

その他

  • find_wy (CrySPY 0.9.0 以上では必須ではなくなった, CrySPY 0.8.x 以下では必要)
